### [FastStone Image Viewer](https://faststone.org/FSViewerDetail.htm)

#### Menu setting

- 设置 → 设置
  - 视图
    - 关联文件打开方式 → 浏览模式
    - 循环 (On)
    - 退出时提示确认 (Off)
    - 覆盖文件时提示确认 (On)
    - 单个文件删除到 → 回收站，不提示
  - JPEG文件
    - 默认JPEG质量 → 100
    - 如果可能，使用原始JPEG文件的质量值 (Off)
    - 颜色缩减取样 → 无（图像品质最好）
  - 外部程序 → 增加 → 目标程序 → 显示名称 `IrfanView` → 确定
