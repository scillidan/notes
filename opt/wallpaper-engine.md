### [Wallpaper Engine](https://wallpaperengine.io/)

#### Usage

1. 新建 → ... → 关闭
2. 编辑 → 在资源管理器中打开 → .json
3. 文件 → 打开最近的项目 → .json
4. 创意工坊 → 在创意工坊上分享壁纸 → 预览图片 → 导入文件 → 发布更新 → 关闭
