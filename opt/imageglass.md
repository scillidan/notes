# [ImageGlass](https://github.com/d2phap/ImageGlass)

## Setting

- More → Settings
	- Tools → Add:
		```
		Name `ExifGlass - EXIF metadata viewer`
		Excutable `ExifGlass.exe`
		Hotkeys `x`
		```
	- Edit → Image editing apps → Add:
		```
		File extension `.jpg;.png;.svg`
		App name `Photoshop`
		Executable `Photoshop.exe`
		```
