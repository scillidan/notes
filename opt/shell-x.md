# [Shell-X](https://github.com/oleg-shilo/shell-x)

````{tab} From releases
1. Download `shell-x.*.7z` ffrom [Releases](https://github.com/oleg-shilo/shell-x/releases).
2. Decompress it to `shell-x\`.
3. Add `shell-x\` into `PATH`.
````

````{tab} scoop
```sh
scoop install shell-x
```
````

```sh
shell-x -r
shell-x -init
```
