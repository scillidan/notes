### [Everything](https://www.voidtools.com/)

#### Setting

- 选项 → 索引/排除列表 → 启用排除列表 → 添加筛选器 → `node_modules\`, `public\`, `site\`, `Rubbish\`
  - 添加文件夹 → 文件夹 → `C:\$Recycle.Bin` → 确定
- 选项 → 常规 → 结果 → 双击路径列打开目录 On
- 索引 → 索引最近变化 Off
- 索引 → NTFS → 自动... Off → 不搜索的本地磁盘 → 包含到数据库 Off, 启用USN日志 Off
