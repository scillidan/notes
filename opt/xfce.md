### [Xfce](https://www.xfce.org/)

#### Setting

- Screensaver → Enable Screensaver (Off)
- Lock Screen → Enable Lock Screen (Off)
- Storage → Removable Storage
  - Mount removable drives when hot-plugged (On)
  - Mount removable media when inserted (On)
