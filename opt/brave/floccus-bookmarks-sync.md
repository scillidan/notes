### [floccus bookmarks sync](https://github.com/floccusAddon)

#### Usage

1. 打开 → ... →  新建账户 → WebDav分享
  ```
  WebDAV URL `URL`
  用户名 `User`
  密码 `Password`
  ```
2. 下一步
  ```
  服务器详细信息 → 书签路径 `floccus/bookmarks.xbel`
  文件夹映射 → 本地文件夹 `/书签栏/`
  同步间隔 `1d`
  嵌套账户 `在其他帐户同步中包括此帐户的本地`
  文件夹 → 保存
  ```
3. 打开 ... WEBDAV → 行为(推送) → 自动同步 (Off)
