### [Langflow](https://github.com/langflow-ai/langflow)

```sh

```

````{tab} Docker Compose
```sh
mkdir langflow

wget https://github.com/langflow-ai/langflow/blob/main/docker_example/docker-compose.yml
sudo docker compose up -d
```
````
