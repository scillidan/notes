````{tab} From source
```sh
git clone --depth=1 https://github.com/yoksel/url-encoder
cd url-encoder
npm install
npm run build
npm run start
```
````

````{tab} PM2
```sh
pm2 serve build 4321 --name url-encoder --spa
```
````

Visit `localhost:4321`.
