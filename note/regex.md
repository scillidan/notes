### Regex

```
.
	any character except a newline
*
	0 or more repetitions
+
	1 or more repetitions
?
	0 or 1 repetitions
{m}
	m repetitions
{m,n}
	m-n repetitions
[]
	set of characters
[^]
	complementing the set
\d
	decimal digit
\D
	not a decimal digit
\s
	whitespace characters (Tab, Enter)
\S
	not a whitespace characters
\w
	word character ... as well as numbers and the underscore
\W
	not a word character
```
