````{tab} From source
```sh
git clone --depth=1 https://github.com/seleb/sketch-to-lineart
cd sketch-to-lineart
npm install
npm run build
serve -s docs -p 4321
```
````

If app take up `4321` port, open `chrome://serviceworker-internals/?devtools` and unregister it.
