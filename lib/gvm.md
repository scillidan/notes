#### [gvm](https://github.com/olimpias/gvm) (Cache)

1. Get `gvm.windows.amd64.exe.zip` from [Releases](https://github.com/olimpias/gvm/releases).
2. Decompress it to `gvm\`.

Get versions list from [All releases](https://go.dev/dl/).

```sh
gvm dl 1.21.0
gvm use 1.21.0
```
