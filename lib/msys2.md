# [MSYS2](https://www.msys2.org/) (Cache)

```sh
pacman -Syyu
pacman -S mingw-w64-ucrt-x86_64-gcc
pacman -S mingw-w64-x86_64-cargo-c mingw-w64-x86_64-protobuf
cargo install atuin
pacman -S mingw-w64-ucrt-x86_64-pkg-config
```
